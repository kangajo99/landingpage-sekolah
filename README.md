# Landing Page Sekolah - Sistem Portofolio Template

Ini adalah repositori sumber kode untuk Landing Page profil sekolah (Lembaga Pendidikan). Proyek ini dikembangkan menggunakan **Tailwind CSS v4** dengan arsitektur modular menggunakan _Custom Web Components_ agar mudah dikelola dan dikembangkan.

## 🚀 Fitur Utama

- **Komponen Modular:** Header menggunakan `customElements` (JavaScript Class) sehingga memudahkan pembaruan navigasi di banyak halaman sekaligus.
- **Responsif:** Desain dioptimalkan untuk perangkat mobile dan desktop.
- **Optimasi Performa:** Menggunakan teknik _Hardware Acceleration_ untuk mencegah _glitch_ pada perangkat seluler.
- **Sistem Navigasi Dinamis:** Mendukung sistem sub-folder (siap untuk katalog multi-template).
- **Pendaftaran PPDB:** Mendukung pendaftaran ppdb via WA bisa untuk iklan.

## 🛠️ Teknologi yang Digunakan

- **HTML5**
- **Tailwind CSS v4** (via CDN)
- **Lucide Icons**
- **JavaScript (ES6)**

## 📂 Struktur Folder

```text
├── images/             # Aset gambar & logo
├── fasilitas.html      # Halaman fasilitas
├── galeri.html         # Halaman galeri poto kegiatan dan fasilitas
├── guru.html           # Halaman tentang guru
├── header.js           # Komponen navigasi utama
├── index.html          # Halaman beranda
├── prestasi.html       # Halaman prestasi
├── profil.html         # Halaman profil lembaga
├── program.html        # Halaman program sekolah
└── README.md           # Dokumentasi proyek

```

## ⚙️ Cara Pengembangan

1. Pastikan kamu memiliki [VS Code](https://code.visualstudio.com/) dan ekstensi **Live Server**.
2. Clone repositori ini: `git clone https://github.com/kangajo99/landingpage-sekolah.git`
3. Buka folder di VS Code.
4. Klik kanan pada `index.html` -> **Open with Live Server**.
